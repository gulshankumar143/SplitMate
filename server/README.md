# SplitMate Backend

## Setup

1. Copy `.env.example` to `.env`
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start server:
   ```bash
   npm run dev
   ```

## Scripts

- `npm run dev` - start development server with nodemon
- `npm start` - run production server
- `npm run seed` - seed default admin and categories

## API

The API is mounted under `/api` and documentation is exposed at `/api/docs`.
